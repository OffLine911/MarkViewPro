# Application Icon Requirements

## appicon.png

Create an application icon with the following specifications:

### Size
- **Recommended:** 1024x1024 pixels
- **Minimum:** 512x512 pixels
- **Format:** PNG with transparency support

### Design Suggestions
- Use a clean, modern design
- Consider using the letters "MV" or "MVP" as the main element
- Use colors that work well on both light and dark backgrounds
- Include visual elements related to Markdown (e.g., document, text formatting symbols)
- Ensure the icon is recognizable at small sizes (16x16, 32x32)

### Required Files
Place the following file in this directory:
- `appicon.png` - The main application icon

### Platform-specific Icons
Wails will automatically generate platform-specific icons from `appicon.png`:
- Windows: .ico file with multiple sizes
- macOS: .icns file with multiple sizes
- Linux: Various PNG sizes

### Tools
You can create icons using:
- [Figma](https://figma.com)
- [Adobe Illustrator](https://www.adobe.com/products/illustrator.html)
- [Inkscape](https://inkscape.org) (free)
- [GIMP](https://www.gimp.org) (free)
